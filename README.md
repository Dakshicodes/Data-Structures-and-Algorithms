# Data Structures and Algorithms

A curated collection of fundamental **Data Structures and Algorithms** implemented in **C++** for learning, revision, and interview preparation.

This repository serves as my personal reference for commonly used algorithms and data structures that I have implemented while practicing DSA.

---

## 📚 Contents

### Data Structures
- Arrays
- Linked Lists
- Stacks
- Queues
- Trees
- Binary Search Trees
- AVL Trees
- Heaps
- Hashing
- Graphs

### Algorithms
- Binary Search
- Bubble Sort
- Counting Sort
- Kadane's Algorithm
- BFS
- DFS
- Dijkstra's Algorithm
- Bellman-Ford Algorithm
- Kruskal's Algorithm
- Closest Pair of Points
- BFPRT (Median of Medians)
- And many more...

## 🎯 Purpose

This repository is created to:

- Keep implementations organized in one place
- Quickly revise algorithms before interviews and contests
- Understand the intuition behind classic DSA problems
- Maintain clean reference implementations
